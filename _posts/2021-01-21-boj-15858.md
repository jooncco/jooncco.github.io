<cite><b>[BOJ #15858](https://www.acmicpc.net/problem/15858)</b></cite>

<span style="color: #3777e6;">Problem</span>
<div>
    a, b, c 세 정수가 주어질 때,<br/>
    \(a \cdot b / c \)를 \(10^{-6}\)의 precision으로 구하는 문제.
    <ul>
        <li>\( 1 \le a, b, c \le 10^9 \)</li>
    </ul>
</div>
<br/>
<span style="color: #3777e6;">Key Idea</span>
<div>
    double로 계산: 50점.<br/>
    long double로 계산: 75점. ㅋㅋ 재밌는 문제였다<br/>
    결국 각 digit을 소수점 아래 6자리 이상 계산해서 해결.<br/>
    <br/>
    \(Complexity: O(1)\)
</div>

<br/>
<div style="color: #3777e6;">Solution</div>
<div><img src="/assets/images/boj-15858.png"/></div>

```cpp
/**
 * author: jooncco
 * written: 2021. 1. 21. Thu. 22:26:54 [UTC+9]
 **/

#include <bits/stdc++.h>
using namespace std;
typedef long long     ll;

ll a,b,c;

int main() {

    cin >> a >> b >> c;
    ll n= a*b;
    string ans= to_string(n/c);
    ans.push_back('.');
    for (int i=0; i < 20; ++i) {
        n %= c;
        n *= 10;
        if (n < c) ans.push_back('0');
        else       ans.push_back((char)('0'+n/c));
    }
    cout << ans;
}

```
