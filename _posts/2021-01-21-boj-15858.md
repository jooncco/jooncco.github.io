<span style="color: #3777e6;">Key Idea</span>
<div>
    double로 계산하면 50점<br/>
    long double로 계산하면 75점<br/>
    나눗셈을 한 자리씩 계산하도록 직접 구현하면 <b>100점</b><br/>
    <br/>
    \(Complexity: O(1)\)
</div>

<br/>
<div style="color: #3777e6;">Solution</div>
<div><img src="/assets/images/boj-15858-result.png"/></div>

```cpp
/**
 * author: jooncco
 * written: 2021. 1. 21. Thu. 22:26:54 [UTC+9]
 **/

#include <bits/stdc++.h>
using namespace std;
typedef long long     ll;

ll a,b,c;

int main() {

    cin >> a >> b >> c;
    ll n= a*b;
    string ans= to_string(n/c);
    ans.push_back('.');
    for (int i=0; i < 20; ++i) {
        n %= c;
        n *= 10;
        if (n < c) ans.push_back('0');
        else       ans.push_back((char)('0'+n/c));
    }
    cout << ans;
}

```
