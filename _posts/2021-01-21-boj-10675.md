<cite><b>[BOJ #10675](https://www.acmicpc.net/problem/10675)</b></cite>

<span style="color: #3777e6;">Problem</span>
<div>
    출발점(A), 도착점(B), 비행편 정보(10000개의 도시중 몇 개를 비행) N개가 주어질때, 하나의 비행편만 이용해서 도착점에 가기 위한 최소 비용을 구하는 문제.
    <ul>
        <li>비행편 예시: 2 ➜ 10000 ➜ 27 ➜ 7 ➜ 46(도시가 중복되어 나타나진 않음)</li>
        <li>비행편의 일부만 이용 가능 (e.g. 27에서 타고 7에서 내리기)</li>
        <li>한 비행편을 구성하는 도시의 최대 개수: \(1 \le n \le 500\)</li>
        <li>비행편 이용 가격: \(1 \le price \le 10000\) </li>
    </ul>
</div>
<br/>
<span style="color: #3777e6;">Key Idea</span>
<div>
    dijkstra겠지 하고 계속 읽었는데 읽다보니 그냥 조건을 만족하는 최솟값 탐색하면 되는 문제였다.<br/>
    N개의 비행편에 대해서, 각 도시들에서 다음 조건을 검사.<br/>
    <ul>
        <li>여기가 시작점이면, \(flag \leftarrow true\)</li>
        <li>여기가 도착점이고, \(flag == true\)이면 최솟값 업데이트</li>
    </ul>
    최솟값이 처음 초기화값\((\infty)\)이라면 -1, 아니라면 그대로를 출력.<br/>    
    <br/>
    \(Complexity: O(N \cdot n)\)
</div>

<br/>
<div style="color: #3777e6;">Solution</div>
<div><img src="/assets/images/boj-10675.png"/></div>

```cpp
/**
 * author: jooncco
 * written: 2021. 1. 21. Thu. 22:39:49 [UTC+9]
 **/

#include <bits/stdc++.h>
using namespace std;

const int INF= 9999;
int A,B,N;

int main() {

    cin >> A >> B >> N;
    int ans= INF;
    int price, cities;
    for (int plane=0; plane < N; ++plane) {
        cin >> price >> cities;
        bool start= 0, end= 0;
        int here;
        for (int j= 0; j < cities; ++j) {
            cin >> here;
            if (here == A) start= 1;
            if (here == B && start) {
                end= 1;
            }
        }
        if (end) ans= min(ans,price);
    }
    cout << (ans == INF ? -1 : ans);
}

```
